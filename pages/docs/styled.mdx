---
title: 'styled'
description: 'Learn more about how to the `styled` API works in Stitches.'
---

A function to create a component including its styles and variants. It receives:

- `element` or `component`
- `styleObject`
- `variantObject`

```jsx
const Button = styled(
  'button',
  {
    appearance: 'none',
    color: 'white',
  },
  {
    variant: {
      primary: {
        backgroundColor: 'red',
      },
      secondary: {
        backgroundColor: 'blue',
      },
    },
  }
);
```
