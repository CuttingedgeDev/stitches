You can consume your tokens seamlessly as CSS values.

First, setup your [tokens](/docs/tokens):

```jsx
export const { styled, css } = createStyled({
  tokens: {
    colors: {
      loContrast: 'white',
      blue500: 'hsl(206,100%,50%)',
    },
    space: {
      '1': '5px',
      '2': '10px',
      '3': '15px',
    },
    fontSizes: {
      '1': '12px',
      '2': '13px',
      '3': '15px',
      '4': '17px',
    },
    radii: {
      '1': '3px',
      '2': '5px',
      '3': '15px',
      pill: '9999px',
    },
  },
});
```

Then, use them:

```jsx live manual
const Button = styled('button', {
  backgroundColor: 'blue500',
  borderRadius: 'pill',
  boxShadow: '0 5px 15px rgba(0, 0, 0, .3)',
  color: 'loContrast',
  fontSize: '4',
  lineHeight: '1',
  fontWeight: 500,
  paddingTop: '3',
  paddingBottom: '3',
  paddingLeft: '4',
  paddingRight: '4',
  border: '0',
});

render(<Button>Button</Button>);
```
