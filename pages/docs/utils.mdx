You can create your own custom CSS properties:

```jsx
createStyled({
  utils: {
    bc: (config) => (value) => ({
      backgroundColor: value;
    }),
    p: (config) => (value) => ({
      padding: value;
    }),
  },
});
```

Try it out:

```jsx live manual
const Box = styled('div', {
  bc: 'purple500',
  p: '3',
  color: 'hiContrast',
});

render(<Box>Box</Box>);
```
