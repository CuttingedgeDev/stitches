## Create custom utils for a smoother developer experience.

Create your own CSS properties in the `utils` object. Add as many utils as you need.

```jsx
createStyled({
  utils: {
    bc: (config) => (value) => ({
      backgroundColor: value;
    }),
    p: (config) => (value) => ({
      padding: value;
    }),
  },
});
```

> The `config` is your [configuration](/docs/installation#create-your-config-file) and the `value` will be the value passed in via the CSS property.

Try it out:

```jsx live manual
const Box = styled('div', {
  bc: 'purple500',
  p: '3',
  color: 'hiContrast',
});

render(<Box>Box</Box>);
```

### Typescript

You can type the value against a specific scale to improve the developer experience:

```jsx
createStyled({
  utils: {
    mx: (config) => (value: keyof typeof theme['space'] | (string & {})) => ({
      marginLeft: value,
      marginRight: value,
    }),
  },
});
```
