## If you want to contribute to Stitches, this guide explains the process.

### Contributing code

1. Browse the open issues until you find an issue you're interested in.
2. Fork Stitches, then clone it: `git clone https://github.com/YOUR_USERNAME/stitches.git`.
3. Create a new branch: `git checkout -b fix-something`.
4. Run `yarn` to install the packages.
5. Make your changes and commit: git add and git commit
6. Make sure all tests pass.
7. Push your branch: `git push -u origin your-branch-name`.
8. Open a PR to the upstream Stitches repo.
9. Make sure to clearly describe your changes in your PR.
10. We'll review your PR as soon as possible and try to get it merged.
