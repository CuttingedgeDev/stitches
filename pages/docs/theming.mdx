## Set up dark mode in minutes.
You can create new themes that override the [configuration tokens](/docs/tokens):

```jsx live manual
const myTheme = css.theme({
  colors: {
    hiContrast: 'hsl(206,10%,5%)',
    loContrast: 'white',
  },
});

const Box = styled('div', {
  padding: '2',
  backgroundColor: 'hiContrast',
  color: 'loContrast',
});

render(() => (
  <div
    className={myTheme}
    style={{
      display: 'flex',
    }}
  >
    <Box>Box</Box>
    <Box
      css={{
        marginLeft: 2, // => maps to tokens.space[2] (10px)
        marginRight: 2, // => maps to tokens.space[2] (10px)
      }}
    >
      Box
    </Box>
    <Box>Box</Box>
  </div>
));
```

The `darkTheme` const represents a classname, which can be added at any point in your application.

You can add multiple themes, which overrides each other by the nested level you apply them.

### Toggling theme

This is how you can toggle between two themes:

```jsx live manual
const myTheme = css.theme({
  colors: {
    hiContrast: 'hsl(206,10%,5%)',
    loContrast: 'white',
  },
});

const Box = styled('div', {
  padding: '2',
  backgroundColor: 'hiContrast',
  color: 'loContrast',
});

render(() => {
  // start with default theme
  const [theme, setTheme] = React.useState(undefined);

  return (
    <div className={theme}>
      <Box css={{ marginBottom: '2' }}>Box</Box>
      <button onClick={() => setTheme(theme ? undefined : myTheme)}>Toggle theme</button>
    </div>
  );
});
```
