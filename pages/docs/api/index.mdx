## config

## createStyled

A function to instantiate Interop. It receives a [config object]() and returns:

- `styled`: a function to create styled components in React
- `css`: a function to create [themes]() and [SSR styles]()

```jsx
const { styled, css } = createStyled({});
```

## styled

A function to create a component including its styles and variants. It receives:

- `element` or `component`
- `styleObject`
- `variantObject`

```jsx
const Button = styled(
  'button',
  {
    appearance: 'none',
    color: 'white',
  },
  {
    variant: {
      primary: {
        backgroundColor: 'red',
      },
      secondary: {
        backgroundColor: 'blue',
      },
    },
  }
);
```

## css

A function to generate a atomic classnames from a style object. It receives a `styleObject`.

```jsx
const card = css({
  backgroundColor: 'gray',
  border: '1px solid black',
  borderRadius: '6px',
});

({ children }) => <div className="card">{children}</div>;
```

### css.theme

A function to overide the default `tokens`. It receives a`tokenObject`.

```jsx
export const darkTheme = css.theme({
  colors: {
    background: 'black',
    foreground: 'white',
  },
});

({ children }) => <div className={darkTheme}>{children}</div>;
```

### css.getStyles

A function to get the generate CSS on the server, and returns:

- `styles`: the generated CSS based on the HTML string
- `result`: the HTML string provided

```jsx
const { styles, result } = css.getStyles(() => '');
```
