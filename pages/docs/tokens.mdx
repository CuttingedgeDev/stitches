---
title: 'Tokens'
description: 'Learn more about how tokens work in Stitches.'
---

You can define your tokens as part of the `createStyled` [config]().

```jsx
export const { styled, css } = createStyled({
  tokens: {
    colors: {
      background: 'lightgray',
      foreground: 'black',
      primary: 'royalblue',
      secondary: 'tomato',
    },
    space: {
      '0': '0',
      '1': '5px',
      '2': '10px',
    },
    fontSizes: {},
    fonts: {},
    fontWeights: {},
    lineHeights: {},
    letterSpacings: {},
    sizes: {},
    borderWidths: {},
    borderStyles: {},
    radii: {},
    shadows: {},
    zIndices: {},
    transitions: {},
  },
});
```

Tokens can then be seamlessly used in the CSS Objects:

```jsx live manual
const Box = styled('div', {
  padding: '2', // => maps to tokens.space[2] (10px)
  backgroundColor: 'background', // => maps to tokens.colors.background (lightgray)
});

render(
  <div
    style={{
      display: 'flex',
    }}
  >
    <Box>Box</Box>
    <Box
      css={{
        marginLeft: 2, // => maps to tokens.space[2] (10px)
        marginRight: 2, // => maps to tokens.space[2] (10px)
      }}
    >
      Box
    </Box>
    <Box>Box</Box>
  </div>
);
```
