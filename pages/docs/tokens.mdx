---
title: 'Tokens'
description: 'Learn more about how tokens work in Stitches.'
---

You can define your tokens as part of the `createStyled` [config]().

```jsx
export const { styled, css } = createStyled({
  tokens: {
    colors: {
      background: 'lightgray',
      foreground: 'black',
    },
    space: {
      '0': '0',
      '1': '5px',
      '2': '10px',
    },
    fontSizes: {},
    fonts: {},
    fontWeights: {},
    lineHeights: {},
    letterSpacings: {},
    sizes: {},
    borderWidths: {},
    borderStyles: {},
    radii: {},
    shadows: {},
    zIndices: {},
    transitions: {},
  },
});
```

Tokens can then be seamlessly used in the CSS Objects:

```jsx live manual
const Box = styled('div', {
  appearance: 'none',
  backgroundColor: 'background', // => the generated value will be `lightgray`
  padding: '2', // => the generated value will be `10px`
});

render(<Box>Box</Box>);
```
