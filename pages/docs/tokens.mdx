## Customise your tokens, breakpoints and configure your utils

### Tokens

You can define your tokens as part of the `createStyled` [configuration](/docs/configuration).

```jsx
export const { styled, css } = createStyled({
  tokens: {
    colors: {
      background: 'lightgray',
      foreground: 'black',
      primary: 'royalblue',
      secondary: 'tomato',
    },
    space: {
      0: '0',
      $1: '5px',
      $2: '10px',
    },
    fontSizes: {},
    fonts: {},
    fontWeights: {},
    lineHeights: {},
    letterSpacings: {},
    sizes: {},
    borderWidths: {},
    borderStyles: {},
    radii: {},
    shadows: {},
    zIndices: {},
    transitions: {},
  },
});
```

Tokens can then be [seamlessly used](/docs/token-aware-values#tokens-mapping) in the CSS Objects:

```jsx live manual
const Box = styled('div', {
  padding: '$2', // => maps to tokens.space[2] (10px)
  backgroundColor: 'background', // => maps to tokens.colors.background (lightgray)
});

render(
  <div
    style={{
      display: 'flex',
    }}
  >
    <Box>Box</Box>
    <Box
      css={{
        marginLeft: 2, // => maps to tokens.space[2] (10px)
        marginRight: 2, // => maps to tokens.space[2] (10px)
      }}
    >
      Box
    </Box>
    <Box>Box</Box>
  </div>
);
```

You can consume your tokens seamlessly as CSS values.

First, setup your [tokens](/docs/tokens):

```jsx
export const { styled, css } = createStyled({
  tokens: {
    colors: {
      $loContrast: 'white',
      $blue500: 'hsl(206,100%,50%)',
    },
    space: {
      $1: '5px',
      $2: '10px',
      $3: '15px',
    },
    fontSizes: {
      $1: '12px',
      $2: '13px',
      $3: '15px',
      $4: '17px',
    },
    radii: {
      $1: '3px',
      $2: '5px',
      $3: '15px',
      pill: '9999px',
    },
  },
});
```

Then, use them:

```jsx live manual
const Button = styled('button', {
  backgroundColor: '$blue500',
  borderRadius: 'pill',
  boxShadow: '0 5px 15px rgba(0, 0, 0, .3)',
  color: '$loContrast',
  fontSize: '$4',
  lineHeight: '1',
  fontWeight: 500,
  paddingTop: '$3',
  paddingBottom: '$3',
  paddingLeft: '$4',
  paddingRight: '$4',
  border: '0',
});

render(<Button>Button</Button>);
```
