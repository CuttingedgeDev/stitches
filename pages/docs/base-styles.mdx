Base styles are the default styles attached to your component. This is how you can add base styles to your component:

```jsx live manual
const Button = styled('button', {
  backgroundColor: 'royalblue',
  borderRadius: '9999px',
  boxShadow: '0 5px 15px rgba(0, 0, 0, .3)',
  color: 'white',
  fontSize: '17px',
  lineHeight: '1',
  fontWeight: '500',
  paddingY: '12px',
  paddingX: '16px',
  border: '0',
});

render(<Button>Button</Button>);
```

With Stitches, styles are written as an object syntax. This is also known as Style Object.

> Note: Writing CSS as strings is not supported.

## Pseudo-classes and elements

All pseudo-classes and pseudo-elements are supported:

```jsx live manual
const Button = styled('button', {
  backgroundColor: 'royalblue',
  borderRadius: '9999px',
  boxShadow: '0 5px 15px rgba(0, 0, 0, .3)',
  color: 'white',
  fontSize: '17px',
  lineHeight: '1',
  fontWeight: '500',
  paddingY: '12px',
  paddingX: '16px',
  border: '0',

  // Pseudo-class example
  ':hover': {
    backgroundColor: 'white',
    color: 'royalblue',
  },
});

render(<Button>Button</Button>);
```

## Descendant selectors

```jsx live manual
const Button = styled('button', {
  backgroundColor: 'royalblue',
  borderRadius: '9999px',
  boxShadow: '0 5px 15px rgba(0, 0, 0, .3)',
  color: 'white',
  fontSize: '17px',
  lineHeight: '1',
  fontWeight: '500',
  paddingY: '12px',
  paddingX: '16px',
  border: '0',

  // Descendant example
  ' svg': {
    display: 'inline-block',
    width: '16px',
    marginLeft: '6px',
  },
});

render(
  <Button>
    Button
    <svg viewBox="0 0 32 32" fill="none" stroke="currentcolor" strokeWidth="2">
      <path d="M22 6 L30 16 22 26 M30 16 L2 16" />
    </svg>
  </Button>
);
```
