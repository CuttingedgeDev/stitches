## How to override styles in the consumption layer.

Stiches provides a `css` prop for overriding styles easily.

```jsx live manual
const Button = styled('button', {
  backgroundColor: 'royalblue',
  color: 'white',
  fontSize: '13px',
  lineHeight: '1',
  fontWeight: '500',
  border: '0',
  height: '$5',
  paddingLeft: '10px',
  paddingRight: '10px',
});

render(
  <Button
    css={{
      borderRadius: '0',
    }}
  >
    Button
  </Button>
);
```

#### Tokens work

```jsx live manual
const Button = styled('button', {
  backgroundColor: 'royalblue',
  color: 'white',
  fontSize: '13px',
  lineHeight: '1',
  fontWeight: '500',
  border: '0',
  height: '$5',
  paddingLeft: '10px',
  paddingRight: '10px',
});

render(
  <Button
    css={{
      borderRadius: '0',
      ':hover': {
        backgroundColor: '$hiContrast',
        color: '$loContrast',
      },
    }}
  >
    Button
  </Button>
);
```
