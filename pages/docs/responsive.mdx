To build responsive UI's, you first need to define your breakpoints in the `breakpoints` object:

```jsx
export const { styled, css } = createStyled({
  breakpoints: {
    default: (rule) => rule,
    bp1: (rule) => `@media (min-width: 520px) { ${rule} }`,
    bp2: (rule) => `@media (min-width: 900px) { ${rule} }`,
    bp3: (rule) => `@media (min-width: 1200px) { ${rule} }`,
  },
});
```

Once created, you can use them in the Style Objects:

```jsx
const Layout = styled('div', {
  display: 'flex'

  // apply styles to the `default` breakpoint
  default: {
    flexDirection: 'column'
  },

  // apply styles to the `bp1` breakpoint
  bp1: {
    flexDirection: 'row'
  }
});
```

You can also use breakpoints when applying variants:

```jsx live manual
const Button = styled(
  'button',
  {
    backgroundColor: 'white',
    borderRadius: '9999px',
    boxShadow: '0 5px 15px rgba(0, 0, 0, .3)',
    color: 'black',
    lineHeight: '1',
    fontWeight: 500,
    paddingLeft: '15px',
    paddingRight: '15px',
    border: '0',
  },
  {
    size: {
      small: {
        fontSize: '13px',
        paddingTop: '5px',
        paddingBottom: '5px',
      },
      large: {
        fontSize: '17px',
        paddingTop: '10px',
        paddingBottom: '10px',
      },
    },
  }
);

render(<Button size={{ default: 'small', bp2: 'large' }}>Responsive button</Button>);
```
