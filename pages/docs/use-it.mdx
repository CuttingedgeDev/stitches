## You can use the function to start creating and styling components.

You can use the `styled` function to start creating and styling components. This function receives three arguments:

```jsx
const Component = styled(elementOrComponent, baseStyles, variantStyles);
```

> The styled function is part of the [setup step](/docs/setup).

#### Create a styled component

This is how you can create a component with base styles:

```jsx live manual
const Button = styled('button', {
  backgroundColor: 'gainsboro',
  borderRadius: '9999px',
  fontSize: '13px',
  lineHeight: '1',
  fontWeight: '500',
  paddingTop: '10px',
  paddingBottom: '10px',
  paddingLeft: '16px',
  paddingRight: '16px',
  border: '0',
});

render(<Button>Button</Button>);
```

#### Pseudo-class

```jsx live manual
const Button = styled('button', {
  backgroundColor: 'gainsboro',
  borderRadius: '9999px',
  fontSize: '13px',
  lineHeight: '1',
  fontWeight: '500',
  paddingTop: '10px',
  paddingBottom: '10px',
  paddingLeft: '16px',
  paddingRight: '16px',
  border: '0',

  // style pseudo-class
  ':hover': {
    transform: 'translateY(-2px)',
    boxShadow: '0 10px 25px rgba(0, 0, 0, .3)',
  },
});

render(<Button>Button</Button>);
```

#### Descendant selector

```jsx live manual
const Button = styled('button', {
  backgroundColor: 'gainsboro',
  borderRadius: '9999px',
  fontSize: '13px',
  lineHeight: '1',
  fontWeight: '500',
  paddingTop: '10px',
  paddingBottom: '10px',
  paddingLeft: '16px',
  paddingRight: '16px',
  border: '0',

  // style descendant
  ' svg': {
    display: 'inline-block',
    verticalAlign: 'bottom',
    width: '16px',
    marginLeft: '6px',
  },
});

render(
  <Button>
    Button
    <svg viewBox="0 0 32 32" fill="none" stroke="currentcolor" strokeWidth="2">
      <path d="M22 6 L30 16 22 26 M30 16 L2 16" />
    </svg>
  </Button>
);
```

#### Class selector

```jsx live manual
const Button = styled('button', {
  backgroundColor: 'gainsboro',
  borderRadius: '9999px',
  fontSize: '13px',
  lineHeight: '1',
  fontWeight: '500',
  paddingTop: '10px',
  paddingBottom: '10px',
  paddingLeft: '16px',
  paddingRight: '16px',
  border: '0',

  // style class selector
  '&.custom-class': {
    marginLeft: '10px',
  },
});

render(
  <>
    <Button>Button</Button>
    <Button className="custom-class">Button</Button>
  </>
);
```

> In addition to the class selector, you can style all other available basic selectors such as [ID selectors](https://developer.mozilla.org/en-US/docs/Web/CSS/ID_selectors), [attribute selectors](https://developer.mozilla.org/en-US/docs/Web/CSS/Attribute_selectors), [universal selectors](https://developer.mozilla.org/en-US/docs/Web/CSS/Universal_selectors), and [type selectors](https://developer.mozilla.org/en-US/docs/Web/CSS/Type_selectors).

#### Combinator selectors

```jsx live manual
const Button = styled('button', {
  backgroundColor: 'gainsboro',
  borderRadius: '9999px',
  fontSize: '13px',
  lineHeight: '1',
  fontWeight: '500',
  paddingTop: '10px',
  paddingBottom: '10px',
  paddingLeft: '16px',
  paddingRight: '16px',
  border: '0',

  // Ajdacent sibling selector
  '& + button': {
    marginLeft: '10px',
  },

  // Child selector
  '& > button': {
    marginLeft: '10px',
  },
});

render(
  <>
    <Button>Button</Button>
    <Button>Button</Button>
  </>
);
```

#### Refer to another component

```jsx live manual
const Icon = styled('svg', {
  display: 'inline-block',
  verticalAlign: 'bottom',
  width: '16px',
});

const Button = styled('button', {
  backgroundColor: 'gainsboro',
  borderRadius: '9999px',
  fontSize: '13px',
  lineHeight: '1',
  fontWeight: '500',
  paddingTop: '10px',
  paddingBottom: '10px',
  paddingLeft: '16px',
  paddingRight: '16px',
  border: '0',

  // style another component
  [`& ${Icon}`]: {
    marginLeft: '6px',
  },
});

render(
  <Button>
    Button
    <Icon viewBox="0 0 32 32" fill="none" stroke="currentcolor" strokeWidth="2">
      <path d="M22 6 L30 16 22 26 M30 16 L2 16" />
    </Icon>
  </Button>
);
```

#### Style a React component

```jsx live manual
const RightArrow = () => (
  <svg
    className="right-arrow"
    viewBox="0 0 32 32"
    fill="none"
    stroke="currentcolor"
    strokeWidth="2"
  >
    <path d="M22 6 L30 16 22 26 M30 16 L2 16" />
  </svg>
);

// add a `toString` method
RightArrow.toString = () => '.right-arrow';

const Button = styled('button', {
  backgroundColor: 'gainsboro',
  borderRadius: '9999px',
  fontSize: '13px',
  lineHeight: '1',
  fontWeight: '500',
  paddingTop: '10px',
  paddingBottom: '10px',
  paddingLeft: '16px',
  paddingRight: '16px',
  border: '0',

  // style any react component
  [`& ${RightArrow}`]: {
    display: 'inline-block',
    verticalAlign: 'bottom',
    width: '16px',
    marginLeft: '6px',
  },
});

render(
  <Button>
    Button <RightArrow />
  </Button>
);
```

> Remember to add the `toString` method on your component. That needs to return a string matching the desired selector. The selector can be anything you want.
