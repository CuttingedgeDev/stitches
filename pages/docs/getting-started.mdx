Welcome to the Interop documentation!

Interop is a modern CSS-in-JS library. It's built with performance and developer experience in mind.

## Installation

To install Interop, run:

```bash
npm install @interop/react
# or
yarn install @interop/react
```

## Setup

To setup Interop, call the `createStyled` function and export the `styled` and `css` functions provided. This function accepts a config argument.

```jsx
// interop.ts
import { createStyled } from '@interop/react';

const { styled, css } = createStyled({});

export { styled, css };
```

From now on, you'll be relying on the `styled` and `css` functions from here, rather than from the package.

## Usage

To use Interop, import your `styled` function, choose your rendering element and:

```jsx live manual
const Button = styled('button', {
  appearance: 'none',
  backgroundColor: 'tomato',
  color: 'white',
});

render(<Button>Your button</Button>);
```

## Variants

Interop supports variants out-of-the-box:

```jsx live manual
const Button = styled(
  'button',
  {
    appearance: 'none',
    color: 'white',
  },
  {
    variant: {
      primary: {
        backgroundColor: 'tomato',
      },
      secondary: {
        backgroundColor: 'royalblue',
      },
    },
  }
);

render(
  <>
    <Button variant="primary">Primary button</Button>
    <Button variant="secondary">Secondary button</Button>
  </>
);
```
