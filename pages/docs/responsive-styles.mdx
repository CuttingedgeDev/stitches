## How to configure breakpoints and apply variants responsively.

#### Configure your breakpoints

Define your breakpoints in the `breakpoints` object. Add as many breakpoints as you need.

```jsx
export const { styled, css } = createStyled({
  breakpoints: {
    bp1: (rule) => `@media (min-width: 640px) { ${rule} }`,
    bp2: (rule) => `@media (min-width: 768px) { ${rule} }`,
    bp3: (rule) => `@media (min-width: 1024px) { ${rule} }`,
    bp4: (rule) => `@media (min-width: 1280px) { ${rule} }`,
  },
});
```

> Note: Choose whichever naming convention you prefer. We recommend either <Code>bp1</Code>, <Code>bp2</Code>, <Code>bp3</Code>, <Code>bp4</Code> etc. or <Code>sm</Code>, <Code>md</Code>, <Code>lg</Code>, <Code>xl</Code> etc.

#### Responsive variants

Once configured, you can apply different variants at different [breakpoints](/docs/breakpoints). You must use the <Code>initial</Code> breakpoint to declare the initial variant before any breakpoints are applied.

In the example below, we apply the `medium` size variant at the `bp2` breakpoint, and the `large` size variant at the `bp3` breakpoint.

```jsx live manual
const Button = styled('button', {
  alignItems: 'center',
  backgroundColor: 'royalblue',
  border: 'none',
  borderRadius: '9999px',
  color: 'white',
  display: 'inline-flex',
  fontWeight: 500,
  lineHeight: '1',
  paddingLeft: '10px',
  paddingRight: '10px',

  variants: {
    size: {
      small: {
        fontSize: '13px',
        height: '25px',
      },
      medium: {
        fontSize: '15px',
        height: '35px',
        paddingLeft: '15px',
        paddingRight: '15px',
      },
      large: {
        fontSize: '17px',
        height: '45px',
        paddingLeft: '20px',
        paddingRight: '20px',
      },
    },
  },
});

render(<Button size={{ initial: 'small', bp2: 'medium', bp3: 'large' }}>Responsive button</Button>);
```

> Note: If you're using TypeScript, your variants will be typed.

#### Using breakpoints inline

If you wish, you can also use <Code>breakpoints</Code> inside the Style Objects:

```jsx
const Button = styled('button', {
  height: '35px'

  // apply styles to the `bp1` breakpoint
  bp1: {
    height: '45px'
  }
});
```

> Note: In most cases, we do not recommend applying responsive breakpoints inline. Ideally, your component styles should be immutable.
