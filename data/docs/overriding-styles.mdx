---
title: Overriding Styles
description: How to override styles in the consumption layer.
---

Stitches provides a `css` prop for overriding styles easily.

<Preview>
  <DemoButton
    css={{
      borderRadius: '0',
      '&:hover': {
        backgroundColor: 'black',
        color: 'white',
      },
    }}
  >
    Button
  </DemoButton>
</Preview>

```jsx line=3-7
const Button = styled('button', { ...baseStyles });

() => (
  <Button
    __css__={{
      borderRadius: '0',
      '&:hover': {
        backgroundColor: 'black',
        color: 'white',
      },
    }}
  >
    Button
  </Button>
);
```

You can use tokens in the `css` prop.

<Preview>
  <DemoButton
    css={{
      borderRadius: '0',
      '&:hover': {
        backgroundColor: '$blue800',
      },
    }}
  >
    Button
  </DemoButton>
</Preview>

```jsx line=3-7
const Button = styled('button', { ...baseStyles });

() => (
  <Button
    css={{
      borderRadius: '__$1__',
      '&:hover': {
        backgroundColor: '__$blue800__',
      },
    }}
  >
    Button
  </Button>
);
```

You can also use `breakpoints` in the `css` prop.

<Preview>
  <DemoButton
    css={{
      borderRadius: '0',
      color: 'white',
      '&:hover': {
        backgroundColor: '$blue800',
      },
      when: {
        initial: {
          backgroundColor: '$red800',
        },
        bp2: {
          backgroundColor: '$green800',
        },
        bp3: {
          backgroundColor: '$purple800',
        },
      },
    }}
  >
    Button
  </DemoButton>
</Preview>

```jsx line=3-25
const Button = styled('button', { ...baseStyles });

() => (
  <Button
    css={{
      borderRadius: '0',
      color: 'white',
      '&:hover': {
        backgroundColor: '$blue800',
      },
      when: {
        initial: {
          backgroundColor: '$red800',
        },
        bp2: {
          backgroundColor: '$green800',
        },
        bp3: {
          backgroundColor: '$purple800',
        },
      },
    }}
  >
    Button
  </Button>
);
```

#### Overriding the HTML tag

Stitches provides an `as` prop for changing which tag a component outputs.

<Preview>
  <DemoButton as="a" href="#">
    Button
  </DemoButton>
</Preview>

```jsx line=3-7
const Button = styled('button', { ...baseStyles });

() => (
  <Button as="a" href="#">
    Button
  </Button>
);
```
