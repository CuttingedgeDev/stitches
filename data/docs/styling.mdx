#### Object syntax

In order to keep the bundle size to a minimum, Stitches uses JavaScript object notation rather than string syntax. Below, are examples of all of the common use cases.

<Preview>
  <DemoButton>Button</DemoButton>
</Preview>

```jsx
const Button = styled('button', {
  backgroundColor: 'gainsboro',
  borderRadius: '9999px',
  fontSize: '13px',
  border: '0',
});
```

#### Style a React component

<Preview>
  <DemoButton
    css={{
      svg: {
        display: 'inline-block',
        marginLeft: '5px',
        width: '16px',
      },
    }}
  >
    Button
    <svg viewBox="0 0 32 32" fill="none" stroke="currentcolor" strokeWidth="2">
      <path d="M22 6 L30 16 22 26 M30 16 L2 16" />
    </svg>
  </DemoButton>
</Preview>

```jsx line=1-6,10-15,18 id=demo-react-component
const RightArrow = () => (
  <svg __className="right-arrow"__ ... />
);

// add a `toString` method
__RightArrow.toString__ = () => '.right-arrow';

const Button = styled('button', {
  // base styles

  __RightArrow__: {
    display: 'inline-block',
    marginLeft: '5px',
    width: '16px',
  },
});

() => <Button>Button <RightArrow /></Button>;
```

> Remember to add the <h id="demo-react-component" index="2">toString</h> method on your component. That needs to return a string matching the desired selector. The selector can be anything you want.

#### Numeric values

Just like with React, jQuery, and other libraries, Stitches converts numeric values to `px`, if the CSS property doesn't accept unitless values.

```jsx
{
  fontSize: 13,    // `font-size: 13px`
  paddingTop: 10,  // `padding-top: 10px`
  lineHeight: 1,   // `line-height: 1`
  fontWeight: 500, // `font-weight: 500`
}
```

#### Keyframe animations

```jsx line=1-4,8-11
const scaleUp = keyframes({
  '0%': { transform: 'scale(1)' },
  '100%': { transform: 'scale(1.5)' },
});

const Button = styled('button', {
  // base styles

  '&:hover': {
    animation: `${scaleUp} 200ms`,
  },
});
```

#### Global styles

For handling things like global resets, you can write global CSS styles. The `global` function will return another function, which you must call in your app.

```jsx line=1-3,6
const globalStyles = global({
  '*': { margin: 0, padding: 0 },
});

() => {
  globalStyles();

  return <div ... />
};
```
