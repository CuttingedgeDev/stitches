#### Adding variants

<Preview>
  <div style={{ display: 'flex', gap: '20px' }}>
    <DemoButton color="violet">Button</DemoButton>
    <DemoButton color="gray">Button</DemoButton>
  </div>
</Preview>

```jsx line=4-20,23
const Button = styled('button', {
  // base styles

  variants: {
    color: {
      violet: {
        backgroundColor: 'blueviolet',
        color: 'white',
        '&:hover': {
          backgroundColor: 'darkviolet',
        },
      },
      gray: {
        backgroundColor: 'gainsboro',
        '&:hover': {
          backgroundColor: 'lightgray',
        },
      },
    },
  },
});

() => <Button color="violet">Button</Button>;
```

A variant accepts the same style object as the [base styles](/docs/styling).

#### Compound variants

In the case of needing to set styles of a variant, based on a combination of other variants, you can use the `compoundVariants` feature.

<Preview>
  <div style={{ display: 'flex', gap: '20px', alignItems: 'center' }}>
    <DemoButton color="violet" appearance="outline">
      Button
    </DemoButton>
    <DemoButton color="gray" appearance="outline">
      Button
    </DemoButton>
  </div>
</Preview>

```jsx line=13-35,38
const Button = styled('button', {
  ...styles

  variants: {
    color: {
      violet: { ...violetStyles },
      gray: { ...grayStyles },
    },
    appearance: {
      outline: { ...outlineVariants},
    },

    compoundVariants: [{
      color: 'violet',
      appearance: 'outline',
      css: {
        color: 'blueviolet',
        borderColor: 'darkviolet',
        '&:hover': {
          color: 'white',
        },
      }
    },
    {
      color: 'gray',
      appearance: 'outline',
      css: {
        color: 'gray',
        borderColor: 'lightgray',
        '&:hover': {
          color: 'black',
        },
      }
    }]
  },
});

() => <Button color="violet" appearance="outline">Button</Button>
```

#### Default variants

You can use the `defaultVariants` feature to set a variant by default:

```jsx line=11-13,16
const Button = styled('button', {
  ...styles

  variants: {
    color: {
      violet: { ...violetStyles },
      gray: { ...grayStyles }
    },
  },

  defaultVariants: {
    color: 'violet'
  }
});

() => <Button>Button</Button>
```

#### Responsive variants

Once configured, you can apply different variants at different [breakpoints](/docs/breakpoints). You must use the <Code>@initial</Code> breakpoint to declare the initial variant before any breakpoints are applied.

In the example below, we apply `gray` initially, then the `violet` color variant at the `@bp2` condition.

```jsx line=12-16
const Button = styled('button', {
  // base styles

  variants: {
    color: {
      violet: { ...violetStyles },
      gray: { ...grayStyles },
    },
  },
});

() => (
  <Button
    color={{
      '@initial': 'gray',
      '@bp1': 'violet',
    }}
  >
    Button
  </Button>
);
```
